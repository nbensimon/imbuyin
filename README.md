### How do I get set up? ###

Clone down the project
Stay in your root dir before the project dirs start

All commands should be run as root (sudo)

* Create a virtual env

pip install virtualenv

activate the virtualenv, i.e.:

cd virt_imbuyin/
source activate

* Install Django
pip install Django
